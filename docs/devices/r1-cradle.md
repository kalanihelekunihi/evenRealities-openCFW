# Even R1 Charging Cradle — Hardware Specification

The R1 Charging Cradle is a passive accessory that charges the R1 Ring. It contains **no microcontroller, no BLE radio, and no firmware**.

---

## 1. Product Identification

| Property | Value |
|----------|-------|
| **Product Name** | Even R1 Charger / Smart Ring Charger |
| **FCC ID** | 2BFKR-R1C |
| **IC** | 32408-R1C |
| **Type** | Passive charging accessory |

---

## 2. Hardware

| Property | Value |
|----------|-------|
| **Microcontroller** | None |
| **BLE Radio** | None |
| **Firmware** | None |
| **DFU Capability** | None |
| **Charging** | Inductive/contact charging for R1 Ring |
| **Power Input** | USB-C (wired to external power supply) |

The cradle is an electrically passive device — it provides charging power to the R1 Ring via contact or inductive coupling. There is no intelligence, no wireless communication, and no updateable firmware.
